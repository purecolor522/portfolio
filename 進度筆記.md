## data.js 欄位約定（重要）
每件作品是一個物件。常用欄位：
- `category`: `'web'|'dashboard'|'tool'|'edu'` → 進「產品與網頁」區；`'design'` → 進「平面設計」區
- `metric`: 卡片右上的 headline（影響導向，如「1800 檔 < 3 秒掃描」）
- `images: [...]`: 卡片/詳情頁用真實圖（沒給就用 picsum 示意圖 + `seed`）
- `live`: 線上網址 → 詳情頁出現「查看上線版本」按鈕
- `repo`: GitHub 網址 → 詳情頁出現「查看原始碼」按鈕
- `direct: true`: **首頁卡片直接開啟 `live`**（跳過詳情頁）。已上線工具用這個；旗艦案例**不要**設（要讓人讀案例）
- 詳情頁內容：`summary, role, timeline, team, tools, problem, approach[], outcomes[]`
- **旗艦案例專用（work.html 已支援，選填）**：`insight`（洞察段）、`personas[{who,need}]`、`decisions[{choice,why}]`（取捨）、`reflection[]`（回顧）
