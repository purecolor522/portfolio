## 這是什麼
**蘇雅軒 · 產品經理作品集**靜態網站。GitHub: https://github.com/purecolor522/portfolio

- `index.html` — 首頁（Hero、產品作品、平面設計、能力、關於、經歷、聯絡）
- `work.html` — 作品詳情頁模板（一檔服務所有作品，`?id=` 帶入）
- `data.js` — ★ 所有作品與經歷的內容都在這（要改內容改這裡）
- `dist.css` — 預編譯 Tailwind；`images/`、`fonts/` 素材
- 風格：終端機 / coding aesthetic，預設**淺色**（右上可切深色）

## 怎麼本機預覽

```bash
cd 作品集

open http://localhost:8765/index.html
```

## data.js 欄位約定（重要）
每件作品是一個物件。常用欄位：
- `category`: `'web'|'dashboard'|'tool'|'edu'` → 進「產品與網頁」區；`'design'` → 進「平面設計」區；`'event'` → 進「專案統籌」區（活動 PM 案例）
- `metric`: 卡片右上的 headline（影響導向，如「1800 檔 < 3 秒掃描」）
- `images: [...]`: 卡片/詳情頁用真實圖（沒給就用 picsum 示意圖 + `seed`）
- `live`: 線上網址 → 詳情頁出現「查看上線版本」按鈕
- `repo`: GitHub 網址 → 詳情頁出現「查看原始碼」按鈕
- `direct: true`: **首頁卡片直接開啟 `live`**（跳過詳情頁）。已上線工具用這個；旗艦案例**不要**設（要讓人讀案例）
- 詳情頁內容：`summary, role, timeline, team, tools, problem, approach[], outcomes[]`
- **旗艦案例專用（work.html 已支援，選填）**：`insight`（洞察段）、`personas[{who,need}]`、`decisions[{choice,why}]`（取捨）、`reflection[]`（回顧）
